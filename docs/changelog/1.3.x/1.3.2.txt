Version 1.3.2 Changelog
=======================

* Added a dedicated `handshake_enabled` configuration setting to allow enabling or disabling Handshake resolution independently from the BIOS resolver chain.
* Implemented a checkbox toggle for the Handshake daemon in the Resolver Stack tab of `juanita://config`.
* Configured the `hnsd` resolver daemon to be immediately shut down (terminated) if Handshake is disabled during runtime when saving configurations.
* Updated `HandshakeResolver` to skip raw DNS UDP queries and fail immediately when `handshake_enabled` is set to false, bypassing latency overhead.
* Created unit tests validating HandshakeResolver behavior when disabled.
